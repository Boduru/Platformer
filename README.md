# Platformer
Simple C++ 2D Platformer game based on SFML.

## Systems Implemented
- Mutiples Maps
- Map Layers
- Automatic JSON Map File Reader
- Player Gravity
- Player Double & Triple Jumps
- Player Friction and Acceleration
- 2D Collision Detection and Response

 ## Requirements
 - C++
 - SFML
 - Visual Studio 2016
